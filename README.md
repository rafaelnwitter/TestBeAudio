## Running the project
- ### First run the backend
	- #### open the activies-tracker folder 
	- #### run in bash the script
```bash
$ docker-compose up --build
```

- ### Then in another terminal and run the frontend
	- #### open the activies-tracker-frontend folder 
	- #### run in bash the script
```bash
$ npm i && npm start
```
